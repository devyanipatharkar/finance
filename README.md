# Financial Data Extraction Agent

An intelligent AI agent that extracts financial data from Screener.in and exports to Excel & PowerPoint formats.

## Features
- Automated Screener.in login and data extraction
- Financial metrics extraction (P/E, P/B, CAGR, Growth rates)
- Excel export with formatted tables
- PowerPoint export with charts and visualizations
- Batch processing for multiple stocks
- Error handling and retry logic
